# Introduçao a Heranca
Introdução a Herança em PHP

O sistema utiliza herança para os dois tipos de veículos (Carga e Passageiro).
A classe "Veiculo", tem os atributos em comum.  
-A classe "VeiculoCarga" herda os atributos de "Veiculo" e acrescentam a capacidade de carga, exclusiva dessa classe, "VeiculoCarga".  
-A classe "VeiculoPassageiro", herda os atributos de "Veiculo" e retorna que é de passageiro.
